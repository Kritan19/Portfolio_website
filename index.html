About Me

I’ve been a developer for over 4 years, which in "tech years" is basically a century. I originally got into this because I loved the power of command that can make computer do anything we want. I stayed because I’m apparently addicted to the stress of a "learning treadmill" that never stops moving.

The Frontend Obsession

While some people find peace staring at soul-crushing JSON in a dark room, I prefer things that actually move. I’m a frontend-leaning wizard who lives for that sweet dopamine hit of a perfect UI. I build the backend to be solid, but I build the frontend to be magic.

My Vecna Curse

I can architect a complex application and deploy it without breaking a sweat, but naming variables is my personal Vecna Curse. I can fight a Demogorgon of a bug for hours, but the moment I have to name a const, I freeze harder than a Windows update at 99%.

My Developer Spellbook (SKILLS)

1. The "Dopamine" Tier (Frontend)

  • React & Next.js: My primary wands for making things move and look pretty.
  • TypeScript: Because I enjoy the computer yelling at me with red undelines before I even run the code.
  • Tailwind CSS: Transforming "ugly" into "modern" in record time (as long as I don't forget class names).
  • HTML5 & CSS3: The ancient scrolls that started it all.

2. The "Hidden Engine" (Backend)

  • Node.js & Express: Leaving the light of the frontend to battle in the dark arts of servers.
  • Python (Django/Flask): Because sometimes the logic deserves to be readable.
  • Auth (JWT/OAuth): Keeping the White Walkers out of the database.

3. The "Vault" (Databases)

  • PostgreSQL & MySQL: Organizing data into neat little boxes so I don't have to look at messy JSON for too long.
  • MongoDB: For when the boxes are too restrictive and I just want to store "stuff."

4. The "Dark Arts" (DevOps & Tools)

  • Git & GitHub: My personal time machine for when I break something that was working five minutes ago.
  • Docker: Putting a computer inside a computer so your code stops complaining about its environment.
  • AWS & Cloud: Sending my code into the sky and hoping for the best.

5. The "New World" Map (Exploring)

  • AI Agents: Building a digital army to do my work, then praying they don't start hallucinating on the job.
  • Cybersecurity: Learning how to lock the digital doors so tightly that even I get locked out.
  • Healthcare IT: Managing everything from blood samples to X-rays, ensuring the tech is faster than a doctor's handwriting.
